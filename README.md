# Subtitle Bot (자막 생성 및 번역 봇)

이 프로젝트는 영어 영상/음성 파일에서 자동으로 음성을 인식하여 영어 자막(SRT)을 생성하고, 이를 자연스러운 한국어로 번역한 뒤, 최종 MP4 동영상 컨테이너에 영어 및 한국어 소프트 자막(Soft Subtitles) 트랙으로 추가하는 자동화 도구입니다.

---

## 🛠 주요 기능 및 구성 요소

### 1. 음성 인식 및 정렬 (`qwen3_asr_align.py`)
- **Qwen3-ASR-0.6B** 및 **Qwen3-ForcedAligner-0.6B** 모델을 활용하여 음성 인식(STT)과 강제 정렬(Forced Alignment)을 수행합니다.
- 단어 단위의 정확한 타임스탬프를 획득하여 최적의 SRT 자막 파일을 생성합니다.

### 2. 타임스탬프 환각(Hallucination) 방지 번역기 (`translator.py`)
- 기존 번역 방식은 LLM에 타임스탬프가 포함된 SRT 원문을 통째로 전달하여 번역을 수행했기 때문에, LLM이 번역 과정에서 타임스탬프를 미세하게 왜곡하거나 누락시키는 **환각(Hallucination)** 문제가 빈번했습니다.
- 이를 해결하기 위해 다음과 같이 번역 프로세스를 개선하여 **타임스탬프가 변조될 가능성을 원천적으로 차단**했습니다.
  - **SRT 파싱**: Python 코드가 로컬에서 SRT 파일을 파싱하여 자막의 인덱스, 타임스탬프, 텍스트를 구조화된 형태로 분리합니다.
  - **JSON 기반 번역**: 번역이 필요한 영어 텍스트만 모아서 `{"ID": "English Text"}` 형태의 JSON 페이로드로 구성해 LLM에 전송합니다.
  - **JSON 모드 사용**: LLM 호출 시 `response_format={"type": "json_object"}`를 적용하여 정확히 동일한 ID 키를 가진 한글 번역 JSON을 획득합니다. (지원하지 않는 모델의 경우 일반 텍스트 모드로 자동 Fallback 실행)
  - **SRT 재조립**: 번역된 텍스트 데이터를 Python 단에서 로컬 저장되어 있던 원본 타임스탬프 정보와 결합하여 새로운 한글 SRT 파일을 완성합니다.
  - **안전 장치**: 번역 중 특정 자막 키가 유실되더라도 원본 영어 자막을 Fallback으로 사용하므로 자막 전체 싱크가 어긋나는 것을 방지합니다.

### 3. 스트리밍 다운로드 및 합성 파이프라인 (`run.sh`)
- `m3u8` 스트리밍 URL로부터 영상을 다운로드하고, 음성 인식 및 번역 파이프라인을 거쳐 최종 비디오 컨테이너에 영어/한국어 소프트 자막 트랙을 패키징합니다.
- 하드번(Hard-burn) 인코딩 대신 **소프트 자막(Soft Subtitles)** 방식으로 비디오를 재인코딩하지 않고 자막 트랙만 고속으로 주입합니다.
- **안전한 스트림 맵핑**: 원본 비디오에 기존 자막이 포함되어 있더라도 오동작하지 않도록 `-map 0:v -map 0:a?` 옵션을 적용해 원본 비디오/오디오만 명확히 가져오고, 새로 만든 영어/한국어 자막에 대해 정확한 출력 인덱스(`:s:s:0`, `:s:s:1`)를 고정하여 메타데이터 꼬임 문제를 예방합니다.

### 4. 로컬 파일 자막 합성 파이프라인 (`runFromFile.sh`)
- 로컬에 위치한 비디오 파일을 입력받아 `run.sh`와 동일한 흐름으로 영어 자막 생성, 한글 번역, 소프트 자막 트랙 삽입을 수행합니다.
- `run.sh`와 마찬가지로 명시적 스트림 맵핑을 통해 기존 자막과의 충돌을 방지합니다.

## 🚀 시작하기

### 환경 변수 설정
OpenRouter API를 사용하여 번역을 수행하므로, 아래 환경 변수가 터미널 세션에 설정되어 있어야 합니다.

```bash
export OPENROUTER_API_KEY="your-openrouter-api-key"
export OPENROUTER_MODELS="google/gemini-3.1-flash-lite" # 혹은 우선 사용할 모델 목록
```

### 실행 방법

#### 1. 로컬 파일에서 자막 합성하기
```bash
./runFromFile.sh [동영상파일_경로.mp4] ([출력파일명_확장자제외])

# 예시: lecture_01.mp4 파일을 분석하여 영어/한국어 자막이 합성된 lecture_01_final.mp4 생성
./runFromFile.sh lecture_01.mp4
```

#### 2. m3u8 스트리밍 영상에서 자막 합성하기
```bash
./run.sh [출력파일명_확장자제외] "[m3u8_URL]"

# 예시: 스트리밍 다운로드 후 영어/한국어 자막이 합성된 lecture_02_final.mp4 생성
./run.sh lecture_02 "https://example.com/master.m3u8"
```
